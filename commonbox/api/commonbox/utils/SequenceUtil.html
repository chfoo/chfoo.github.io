CommonBox API Documentation

class SequenceUtil

package commonbox.utils

Utility functions for sequences.

Static methods

staticcontains<T> (sequence:BaseSequence<T>, item:T):Bool

Returns whether the given sequence contains the given item.

staticnormalizeIndex<T> (sequence:BaseSequence<T>, index:Int):Int

Returns an index bounded by the sequence range or throw an exception.

Negative indexes are normalized to positive indexes. If the computed index is 0 to length - 1 inclusive, it is returned. Otherwise, OutOfBoundsException is thrown.

staticnormalizeInsertIndex<T> (sequence:BaseSequence<T>, index:Int):Int

Return an index bounded to the sequence range.

Negative indexes are normalized to positive indexes. If the computed index is out of range, it is clamped to 0 or length. The index range is 0 to length inclusive.
