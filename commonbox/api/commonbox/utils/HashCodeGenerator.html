class HashCodeGenerator

package commonbox.utils

Generates hash codes.

A hash code is a uniformly distributed integer value derived from an immutable instance intended for use in optimizing data structures. This is typically used when the object does not have natural ordering.

Hash codes are not unique and different objects can produce the same hash code. Hash codes are only guaranteed to remain the same within the same process. They are not to be used as unique identifiers and should not be shared outside the process.

Static methods

staticgetHashCode (object:Any, ?seed:Int):Int

Returns a hash code for the given object.

Parameters:

object

Immutable instance: null, Bool, Int, Float, String, Equatable, Int64.

seed:

Optional previously generated hash code. This parameter is intended for computing a hash code of multiple objects.
