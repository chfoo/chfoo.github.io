SeedyRNG API Documentation

interface GeneratorInterface

package seedyrng

implemented by GaloisLFSR32, Random, Xorshift128Plus, Xorshift64Plus

Interface for pseudorandom number generators.

Variables

seed:Int64

Initial starting state.

Setting this value will reset the state.

state:Bytes

Current generator state.

The state can be saved and restored to allow resuming a generator.

read onlyusesAllBits:Bool

Whether nextInt returns an integer with a uniform distribution of bits.

When false for example, the generator may output an integer that is never zero or the most significant bit is zero.

Methods

nextInt():Int

Advances the generator and returns an integer.

The returned value is subject to notes in usesAllBits.
