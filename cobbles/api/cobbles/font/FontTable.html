findByCodePoint (codePoint:Int, ?fonts:Array<FontKey>):FontKey

Returns a font that contains the given code point.

If a font cannot be found, the notdefFont font is returned. You can override this behavior by add your own fallback font last. Note that the renderer you choose may use a different behavior such as drawing its own fallback glyph.

Parameters:

codePoint

Unicode code point

searchFonts

If given, the font is selected from the given fonts. Otherwise, fonts are searched in the order that they were added to this instance.
