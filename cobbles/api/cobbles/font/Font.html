Available on all platforms

Opens a font.

Parameters:

path

Filename to a font file.

bytes

If a filename is not given, read the given bytes of a font file.

faceIndex

Selects a font face for unusual font file formats.

setSize (width:Int, height:Int, horizontalResolution:Int, verticalResolution:Int):Void

Sets the font size and resolution (advanced)

Parameters:

width

Width of font in 1/64th of a point. If 0, value of height is used.

height

Height of font in 1/64th of a point. If 0, value of width is used.

horizontalResolution

Horizontal resolution in dots per inch. If 0, value of verticalResolution is used.

verticalResolution

Vertical resolution in dots per inch. If 0, value of horizontalResolution is used.
