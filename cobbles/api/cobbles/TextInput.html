Cobbles API Documentation

class TextInput

package cobbles

Available on all platforms

Unified interface for building text layout by runs of text.

Constructor

new (?textConfig:TextConfig)

Parameters:

textConfig

Optional text config. If not provided, the singleton will be used.

Variables

alignment:Alignment

Indicates the text alignment or justification.

See Layout.

color:Int

Default text color in ARGB format.

read onlyconfig:TextConfig

Text config instance.

font:FontKey

Default font.

fontSize:Float

Default font size in points.

language:String

Default language as a BCP 47 tag.

See TextProperties.

read onlylayout:Layout

Text layout instance.

lineBreakLength:Int

Maximum length of a line in pixels.

If 0 or negative, there is no automatic line breaking.

lineSpacing:Float

The default line spacing as a multiplier of the font height.

orientation:Orientation

The orientation and flow direction of lines.

See Layout.

script:String

Default script of the text as a ISO 15924 tag.

See TextProperties.

read onlyshaper:Shaper

Text shaper instance.

textDirection:Direction

Default visual ordering of the characters in the text.

See TextProperties.

read onlytextSource:TextSource

Input text source instance.

Methods

addInlineObject (inlineObject:InlineObject):Void

Appends an inline object.

addLineBreak (spacing:Float = 1.2):Void

Appends a line break.

Parameters:

spacing

Multiplier of the default font size.

addMarkup (text:String):Void

Parses the markup and appends the text.

The markup is parsed using the MarkupParser that is set on config.

Parameters:

text

Text containing markup.

addText (text:String):TextInputTextFI

Appends a segment of text containing the same properties.

Parameters:

text

Text.

Returns:

A fluent interface for applying text properties that different from the default.

clearText ():Void

Clear the text so that new text may be added.

layoutText ():Void

Perform line breaking, text shaping, and text layout.

The layout will be ready for use in a renderer.
