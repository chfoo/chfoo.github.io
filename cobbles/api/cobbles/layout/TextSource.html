Cobbles API Documentation

class TextSource

package cobbles.layout

Available on all platforms

Collects and stores input text and properties.

Constructor

new (?bidiAlgorithm:BidiAlgorithm, ?lineBreakingAlgorithm:LineBreakingAlgorithm)

Parameters:

bidiAlgorithm

Optional bidirectional algorithm. If provided, it is applied automatically to text. The text direction should be set to match the algorithm's output direction.

lineBreakingAlgorithm

Optional line breaking algorithm. If provided, it is automatically used to break lines when the text contains line breaking characters. Otherwise, use addLineBreak() manually for line breaks.

Unicode code points of the input text.

Default text properties.

Parameters:

relativeSpacing

A multiplier of the default font height that represents the height of the line.
