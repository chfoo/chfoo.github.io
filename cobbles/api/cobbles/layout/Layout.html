Variables

alignment:Alignment = Alignment.Start

Indicates the text alignment or justification.

  • For horizontal orientation, this indicates flush left, centered, or flush right.
  • For vertical orientation, this indicates top alignment, middle alignment, or bottom alignment.

read onlyboundingHeight:Int = 0

The maximum height of the layout containing content in 1/64th of a point.

read onlyboundingWidth:Int = 0

The maximum width of the layout containing content in 1/64th of a point.

lineBreakLength:Int = 0

Maximum length of a line in 1/64th of a point.

This is used for line breaking (also known as word wrapping).

If 0 or negative or line breaking rules is not provided in the text source, there is no automatic line breaking.

read onlylines:Array<LayoutLine>

The text processed into lines.

orientation:Orientation = HorizontalTopBottom

The orientation and flow direction of lines.

  • If horizontal, a line of text will flow from left to right and subsequent lines will flow toward the bottom.
  • If vertical left-right, a line of text will flow from top to bottom and subsequent lines will flow toward the right.
  • If vertical right-left, a line of text will flow from top to bottom and subsequent lines will flow toward the left.

relativeLineSpacing:Float = 1.2

The default line spacing as a multiplier of the font height.

read onlyresolution:Int = 72

Font resolution in dots per inch.

read onlytextSource:TextSource
