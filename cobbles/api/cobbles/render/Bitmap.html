Cobbles API Documentation

interface Bitmap

package cobbles.render

implemented by BaseBitmap

Available on all platforms

Interface for drawing glyphs to bitmaps.

Methods

drawBytes (x:Int, y:Int, width:Int, height:Int, bytes:Bytes):Void

Draws to the bitmap using the given bitmap.

Parameters:

x

Position on x-axis in pixels.

y

Position on y-axis in pixels.

width

Width of the given bitmap in pixels.

height

Height of the given bitmap in pixels.

bytes

The grayscale, 8 bits per pixel, bitmap to be drawn.

drawDebugBox (x:Int, y:Int, width:Int, height:Int, filled:Bool):Void

Draw rectangle used for debugging.

Parameters:

x

Position on x-axis in pixels.

y

Position on y-axis in pixels.

width

Width of the rectangle in pixels.

height

Height of the rectangle in pixels.

filled

Whether to draw something in the middle to distinguish the rectangle from a glyph of a square. Such examples include an "X" or "?" symbol. It does not mean to flood fill.

getHeight ():Int

Returns the height of the bitmap in pixels.

getWidth ():Int

Returns the width of the bitmap in pixels.
