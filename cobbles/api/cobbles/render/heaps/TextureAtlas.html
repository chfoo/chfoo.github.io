Cobbles API Documentation

class TextureAtlas

package cobbles.render.heaps

@:directlyUsed

Available on all platforms

A texture builder that contains glyphs required to display text.

Constructor

new(width:Int, height:Int)

Parameters:

width

Width in pixels of the texture. Should be a multiple of 2.

height

Height in pixels of the texture. Should be a multiple of 2 and match width

Variables

read onlybitmap:PixelsBitmap

read onlyheight:Int

read onlytexture:Texture

read onlywidth:Int

Methods

addGlyph(tile:TileInfo, glyph:GlyphInfo):Void

Add glyph to the texture and draw to the texture pixels buffer.

clearGlyphs():Void

Removes all the glyphs.

This does not affect the texture until it is rebuilt.

clearTexture():Void

Clear the texture pixels buffer.

getGlyphTile(glyphKey:GlyphID):GlyphTile

Returns a new Tile instance that is positioned to a glyph in the texture.

hasGlyph(glyphKey:GlyphID):Bool

Returns whether the texture atlas has the given glyph.

Parameters:

null

glyphKey

uploadTexture():Void

Upload the texture pixels buffer to the texture.
