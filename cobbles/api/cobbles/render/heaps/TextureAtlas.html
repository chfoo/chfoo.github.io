Cobbles API Documentation

class TextureAtlas

package cobbles.render.heaps

Available on all platforms

A texture builder that contains glyphs required to display text.

Constructor

new (width:Int, height:Int, maxGlyphs:Int = 1024)

Parameters:

width

Width in pixels of the texture. Should be a multiple of 2.

height

Height in pixels of the texture. Should be a multiple of 2 and match width

maxGlyphs

If positive, maximum number of glyphs where least recently used glyphs are removed before adding new ones. If 0, glyphs are not automatically removed.

Variables

read onlybitmap:PixelsBitmap

read onlyheight:Int

read onlytexture:Texture

read onlywidth:Int

Methods

addGlyph (glyphKey:GlyphRenderKey, glyphBitmap:GlyphBitmap):Void

Adds a glyph to be drawn.

buildTexture ():Void

Draws the glyphs to the texture.

clear ():Void

Removes all the glyphs.

This does not affect the texture until it is rebuilt.

getGlyphTile (glyphKey:GlyphRenderKey):GlyphTile

Returns a new Tile instance that is positioned to a glyph in the texture.

hasGlyph (glyphKey:GlyphRenderKey):Bool

Returns whether the texture atlas has the given glyph.
