Cobbles API Documentation

class TileGroupRenderer

package cobbles.render.heaps

extends BaseRenderer

Available on all platforms

Renders glyphs by using TileGroup instances.

A texture atlas is used to store glyphs and Tile instances are created whenever a glyph needs to be displayed on the screen.

Constructor

new (fontTable:FontTable, ?textureAtlas:TextureAtlas, glyphCacheSize:Int = 1024)

Parameters:

fontTable

Font table containing the required fonts.

textureAtlas

If given, it will be reused. Otherwise, a new texture atlas is used.

glyphCacheSize

The cache size for the number of glyph bitmaps cached from the glyph rasterizer.
