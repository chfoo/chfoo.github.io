Callfunc API Documentation

abstract AnyInt(Dynamic)

package callfunc

from Dynamic to Dynamic

Available on all platforms

Abstract over Dynamic for integer types not known at compile time.

This abstract contains convenient methods to convert dynamic objects to Int and Int64 at runtime with optional data loss checking. It encapsulates the if-else type checking.

It is used in places where you need to check the type of the value at runtime or as a substitute for an "either" type in functions that accept either Int or Int64.

Methods

@:value({ checkTruncation : false })@:impltoInt (this:Dynamic):Int

Converts value to Int.

If the value is Int, it is returned unchanged. If it is Int64, the value will be converted or truncated with data loss.

Parameters:

checkTruncation

If true, an exception will be thrown if the value cannot be converted with data loss.

Throws:

String

If the value is not Int or Int64, or value cannot be converted from Int64 without data loss.

@:impltoInt64 ():Int64

Converts value to Int64.

If the value is Int it is promoted to Int64. If the value is Int64, it is returned unchanged.

Throws:

String

If the value is neither Int or Int64.
