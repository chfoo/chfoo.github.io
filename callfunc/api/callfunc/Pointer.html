interface Pointer

package callfunc

implemented by EmPointer, PointerImpl

Available on all platforms

Represents a C pointer.

Variables

The value of the pointer that represents an address in memory where the targeted data is stored.

Memory instance of which this pointer belongs to.

@:value({ offset : 0 })get (dataType:DataType, offset:Int = 0):Any

Returns the value at the addressed memory location.

Parameters:

dataType

Data type of the value expected at the addressed memory.

offset

Value in bytes used to offset the address. This is used to access fields in a struct.

Returns:

A value converted to the type Int, haxe.io.Int64, Float, or Pointer.

Integer data types that fit within 32 bits will be promoted to Int while wider integers will be promoted to haxe.io.Int64.

isNull ():Bool

Returns whether the address does not point to anywhere.

Sets the value at the address memory location.

Parameters:

value

A value of type Int, haxe.io.Int64, Float, or Pointer. Numeric types will be promoted and casted appropriately.

dataType

Data type of the value expected at the addressed memory.

offset

Value in bytes used to offset the address. This is used to access fields in a struct.
