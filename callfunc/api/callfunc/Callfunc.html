@:value({ terminator : true, encoding : UTF8 })allocString(text:String, encoding:Encoding = UTF8, terminator:Bool = true, ?lengthCallback:Int‑>Void):Pointer

Converts a Haxe string to a new C string and returns its pointer.

The C string will be encoded with the given encoding with a null terminator.

This is equivalent of determining the number of bytes in the encoded string, allocating memory, and writing the encoded string with the pointer.

The caller is responsible for freeing the string.

Parameters:

text

String to be encoded,

encoding

Encoding such as UTF-8.

terminator

Whether to include a null-terminator.

lengthCallback

A callback that accepts the number of bytes allocated

bytesToPointer(bytes:Bytes):Pointer

Available on cpp, hl

Exposes a pointer to the underlying C array of a Haxe Bytes.

This method is used to share from a Haxe array to a C library.

Care must be ensured that the Bytes instance has not been garbage collected when using the pointer.

This method is not portable across targets.

Parameters:

null

bytes
