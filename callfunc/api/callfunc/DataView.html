Callfunc API Documentation

interface DataView

package callfunc

implemented by BytesDataView

Available on all platforms

Provides an abstract view to Bytes.

The get and set integer methods use signed/unsigned N-bit values. Float and double numbers are 32- and 64-bit values respectively. The position is defined as the offset from the start of the view.

Protection against out-of-bounds is not guaranteed and is not mandatory for implementations.
