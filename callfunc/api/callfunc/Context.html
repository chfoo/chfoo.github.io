Methods

newCallback (haxeFunction:Array<Any> ‑> Any, ?params:Array<DataType>, ?returnType:DataType):Callback

Returns a Callback instance for passing Haxe functions to C code.

Availability depends on the libffi platform support for closures.

Parameters:

haxeFunction

Callback function to be wrapped that accepts an array of arguments or an empty array.

params

Data types corresponding to the function parameters exposed to the C code.

returnType

Data type of the return value of the exposed function.

Throws:

String

An error message if the data type is invalid or any other error.

newLibrary (name:String):Library

Returns a library handle to a dynamic library.

Parameters:

name

A filename to the dynamic library as accepted by dlopen() or LoadLibrary() on Windows.

Throws:

String

An error message if the library could not be found or any other error.
