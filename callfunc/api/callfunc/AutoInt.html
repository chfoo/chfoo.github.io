abstract AutoInt(Int)

package callfunc

from Int, to Int

Available on all platforms

Int abstract with automatic truncation of Int64.

Int64 will be converted to Int with possible data loss.

This abstract can be used for implicit casing between Int64 as "syntactic sugar".
