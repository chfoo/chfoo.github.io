LightningHx API documentation

abstract WriteFlags(Int)

package lightninghx

to Int,

Write options.

Variables

inline read onlyAppend:WriteFlags = 0x20000

Data is being appended, don't split full pages.

inline read onlyAppendDup:WriteFlags = 0x40000

Duplicate data is being appended, don't split full pages.

inline read onlyCurrent:WriteFlags = 0x40

put: Overwrite current key-data pair.

inline read onlyMultiple:WriteFlags = 0x80000

Store multiple data items in one call. Only for MDB_DUPFIXED.

inline read onlyNoDupData:WriteFlags = 0x20

put: Don't write if key-data pair exists. cursor delete: Remove duplicates.

inline read onlyNoOverwrite:WriteFlags = 0x10

put: Don't write if key exists.

inline read onlyReserve:WriteFlags = 0x10000

put: Reserve space for data and return a pointer to the space.

Don't use this in Haxe.

Methods
