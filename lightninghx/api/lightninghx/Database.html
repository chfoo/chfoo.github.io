LightningHx API documentation

interface Database

package lightninghx

implemented by CPPDatabase, SafetyDatabase

A handle to key-value data table.

A database belongs to a transaction, however, the database is usually reused among transactions. To use this pattern in an object-oriented manner, call the reuse() method.

The database can be shared among threads but the methods are not thread-safe except for reuse().

Methods may throw an exception excluding close(), reuse() and the comparison functions.

Methods

close ():Void

Release resources held by this instance.

Calling this method is usually unnecessary.

Do not call this method if there is any open cursors or uncommitted transactions.

This method is not thread safe. This instance must not be used after calling this method.

compareKey (keyA:Bytes, keyB:Bytes):Int

Return the ordering between two keys.

compareValue (valueA:Bytes, valueB:Bytes):Int

Return the ordering between two values in a DUPSORT database.

delete (key:Bytes, ?data:Bytes):Void

Remove items.

Parameters:

key

Key to search.

data

Data to search if DUPSORT. If there is no matching item, an exception is thrown.

drop (delete:Bool = false):Void

Empty or delete the database.

This method will delete all the keys. If delete is true, then the database itself will be deleted and closed.

Parameters:

delete

Whether to delete and close the database.

get (key:Bytes):ReadOnlyBytes

Return data for the given key.

The data returned is a reference to the memory map and must not be modified.

Parameters:

key

Key used to search.

getFlags ():Flags<DatabaseFlags>

Return the database options.

openCursor ():Cursor

Open a new cursor.

put (key:Bytes, data:Bytes, ?flags:Flags<WriteFlags>):Void

Store the key-data pair.

Parameters:

key

Key of the pair.

data

Data of the pair.

flags

Write options. The default behavior is to replace the existing pair if it exists.

reuse (transaction:Transaction):Database

Return new instance associated with the given transaction.

This method is used to share and reuse databases without releasing the underlying database resource and handle.

Parameters:

transaction

Any other transaction.

stat ():Statistics

Return statistics about the database.
