run (callback:Void ‑> T, ?scheduler:Scheduler):Void

Runs synchronous code as a task.

When an appropriate implementation is provided, this method runs synchronous code in the background. By default, the callback is run synchronously.

Parameters:

callback

A function returning the result for the task.

scheduler

Optional scheduler to override the default scheduler.
