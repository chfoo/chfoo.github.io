Resdb API Documentation

class ResourceHelper

package resdb

Helper class that uses haxe.Resource as its resource mechanism.

Note the names do not have to be a filename but they still need to be unique. The names are used as a prefix with the page number added as a suffix. For example, the name example will generate resources such as example-0 or example-123.

Static methods

staticaddResource (pagePacker:PagePacker):Array<Bytes>

Adds the records to the resource from a page packer.

This is intended to be called inside an initialization macro.

Parameters:

pagePacker

A page packer containing the records.

staticaddResourcePages (name:String, dataPages:Array<Bytes>):Void

Adds the pages to the resource.

This is intended to be called inside an initialization macro.

Parameters:

name

Name prefix.

dataBytes

Raw pages produced by a page packer.

staticgetDatabase (config:DatabaseConfig, pageCache:Int = 8):Database

Returns the database using the given resource name.
