Textball API Documentation

class Textball

package textball

Convenience methods for using the global Translator.

Static variables

statictranslator:Translator

Current global Translator instance.

Static methods

staticgetString (id:Int):String

Returns the localized text for the given string ID.

See Translator.getString.

staticgetStringPlural (id:Int, count:Int):String

Returns the localized plural form text for the given string ID and number.

See Translator.getStringPlural.

staticgetText (text:String):String

Returns the localized text for the given text.

See Translator.getText.

staticgetTextPlural (singular:String, plural:String, count:Int):String

Returns the localized plural form text for the given text and number.

See Translator.getTextPlural.
